# 3D Mechanical Design: Discord Logo Keychain (Onshape)

This project fulfills **Mechanical Task 2** for the training program. It features a 3D CAD model of the iconic **Discord Logo** designed from scratch using **Onshape**, prepared for 3D printing as a custom keychain accessory.

---

## 🛠️ CAD Modeling Workflow

1. **Reference & Sketching:**
   * Traced the outer boundary of the Discord logo using precise Splines and Degree-2 curves to match the iconic geometry.
2. **Keyring Hole Cutout:**
   * Created circular cutouts for the eyes/keyring attachment with a diameter of $\varnothing 25\text{ mm}$ and a $25\text{ mm}$ offset from the symmetry line.
3. **Extrusion (3D Feature):**
   * Applied a solid **Extrude of $2\text{ mm}$** depth to generate the 3D plate thickness, ensuring a balance between light weight and structural durability.
4. **File Export:**
   * Exported the completed 3D model into **.STL format** for direct 3D printing and slicing.

---

## 📐 Design Specifications

* **CAD Software:** Onshape
* **Base Thickness (Extrude):** $2\text{ mm}$
* **Eye Cutout Diameter:** $\varnothing 25\text{ mm}$
* **File Format:** `.STL`
